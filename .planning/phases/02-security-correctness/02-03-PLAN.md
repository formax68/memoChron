Create a new file `src/utils/errors.ts`. The file has NO imports. It exports a single named function:

```
/**
 * Extract a printable message from an unknown thrown value.
 * Use this in every catch block to avoid "[object Object]" or "undefined"
 * outputs when the thrown value is not an Error instance.
 * @param err Unknown caught value (Error, string, number, custom throwable, ...)
 * @returns The Error.message if err instanceof Error, otherwise String(err)
 */
export function errorMessage(err: unknown): string {
  return err instanceof Error ? err.message : String(err);
}
```

Notes:
- No imports.
- No additional helpers in this file (single-purpose, per D-09).
- No file-level constants.
- Use 2-space indentation, double quotes, trailing commas — per CONVENTIONS.md.
- The function does NOT unwrap `error.cause`; that is explicitly deferred (CONTEXT.md "Deferred Ideas").
